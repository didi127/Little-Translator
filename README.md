# Little-Translator


Little Translator是一个基于React + TypeScript  + vite， 通过 百度翻译Api 制作的可以实现 多种语言 翻译到 中文的项目

#### 🛠Development Setup

```
## 安装依赖
npm install

## 启动项目
npm run dev
```

![预览图](https://github.com/didi127/Little-Translator/blob/develop/src/assets/image/viewimg.png)
